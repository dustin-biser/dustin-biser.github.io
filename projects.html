2D Rigid Body Physics Engine

My senior project at the University of Maryland in Spring 2012.
My advisor for this project was Prof. David Mount.

The physics engine uses impulse-based dynamics for collision response (an idea pioneered by Brian Miritch) due to the fact that different types of contact (colliding, rolling, sliding, and resting) can all be handled similarly within the same framework.

Rigid bodies are dynamically simulated using an iterative time stepping scheme (Runge-Kutta Method) in order to solve the equations of motion.

Collision detection is handled using axis aligned bounding boxes for broad phase detection, in combination with separating axis tests for narrow phase detection. If collision is found between two bodies, relative velocities are computed at the points of contact, and impulses are applied at these points to prevent penetration.

In order for users to interact with bodies during a simulation, dampened spring forces are computed on the fly in response to mouse inputs.

Check out the project's Github Repository
