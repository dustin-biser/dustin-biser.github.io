Eulerian Grid-Based Smoke Simulation

This is a grid-based method for simulating smoke. Semi-Lagrangian advection was used with an RK2 backtrace and bilinear interpolation. This work is based off the method's introduced by Jos Stam in his paper Stable Fluids.

Source Code

Reference
Stam, Jos. "Stable fluids". Proceedings of the 26th annual conference on Computer graphics and interactive techniques. ACM Press/Addison-Wesley Publishing Co., 1999.

Shadowing Mapping

Implementation of the Shadow Mapping algorithm.


Depth Buffer offsets were used to prevent shadow surface acne.
Percentage Closer filtering was applied to the shadow map in order to anti-alias shadow edges.

Quaternion Based Camera

Demo of a quaternion backed camera implementation.

The pose of the camera is computed and stored within a quaternion.

View matrix calculations are computed within the Camera class via lazy evaluation to increase performance.

After a set number of rotations, the camera automatically renormalizes it's local coordinate system to prevent numerical error accumulation.
